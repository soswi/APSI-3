# APSI-3 🗺️📌
Web application for automatic planning and keeping track of walking routes.

The routes generated by the app are based on user preferences and tailored to create a "natural" walking experience. 

# User preferences
Currently the preferences supported by the app are:
- air quality
- amount of noise
- proximity of greenery

# Data sources
Data used to create the walking routes is collected from:
- [Openstreetmap](https://www.openstreetmap.org)
- [Warsaw WMS (Web Map Service)](https://wms.um.warszawa.pl/)
- [Warsaw Open Data API](https://api.um.warszawa.pl)
